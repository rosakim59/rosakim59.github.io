왜 x + a 대신
x − a를 넣을까?

억지로 외워야 했던 그 공식, 사실 지극히 논리적이었습니다.
'집합'이라는 작은 관점의 전환으로 모든 것이 명확해집니다.

⏱ 읽는 데 약 5분  ·  🎛 직접 조작하며 확인

🤔 우리가 헷갈렸던 이유

고등학교 수학 시간, 함수의 평행이동을 처음 배울 때 대부분의 학생이 같은 벽에 부딪힙니다. 오른쪽(양의 방향)으로 이동했는데 왜 식에는 더하기가 아니라 빼기를 써야 하는가— 이 직관과의 충돌이 공식을 '이해'가 아닌 '암기'의 대상으로 전락시킵니다.

교과서의 설명

"함수 $y = f(x)$의 그래프를 $x$축 방향으로 $a$만큼, $y$축 방향으로 $b$만큼 평행이동하면, $x$ 대신 $x - a$를, $y$ 대신 $y - b$를 대입한다."

🤔 "플러스 방향으로 이동했는데 왜 빼기를 쓰지? 직관과 정반대 아닌가?"
이 의문을 품었던 사람이라면, 지금부터 읽어볼 가치가 충분합니다.

📍 그래프는 '선'이 아니라 '점들의 집합'이다

가장 핵심적인 관점의 전환입니다. 매끄러운 곡선처럼 보이는 그래프는, 사실 특정 규칙을 만족하는 무수히 많은 순서쌍 $(x, y)$의 모임, 즉 집합입니다. 이를 집합 기호(조건제시법)로 정확히 표현하면 아래와 같습니다.

{ (x, y) | y = x2 }

아래 버튼을 눌러 규칙 $y = x^2$을 만족하는 점들을 하나씩 확인해 보세요. 각각의 점이 모여 우리가 아는 포물선이 됩니다.

적용 규칙
y = x2
xy = x²순서쌍

➡️ 점의 이동과 새로운 좌표

이제 슬라이더로 이동 거리를 직접 설정해 보세요. 점 하나를 이동시키는 것은 직관적으로 더하기(+)가 맞습니다. 새로운 위치에 이사 온 점에는 임시로 대문자 $X, Y$(더미 변수)를 붙여 구분합니다.

$x$축 이동 거리 ($a$) 1
$y$축 이동 거리 ($b$) 3
원래 점
(x, y)
→
이동 후 (더미 변수)
(X, Y) = (x + 1, y + 3)

이동한 만큼 더하는 것, 이 단계까지는 자연스럽습니다.

🔑 마이너스(−)의 진짜 정체

우리가 아는 단서는 오직 원래 규칙 $y = x^2$ 뿐입니다. 새 점 $X, Y$의 규칙을 구하려면, 방금 만든 식을 소문자 $x, y$에 대해 이항(移項)해야 합니다. 슬라이더 값에 따라 아래 수식이 실시간으로 갱신됩니다.

Step 1  |  새로운 좌표 정의
X = x + 1
Y = y + 3
⇒
Step 2  |  이항 ← 마이너스의 근원
x = X − 1
y = Y − 3
새 좌표에서 이동량을 빼야 원래 위치가 복원됩니다.
Step 3  |  원래 규칙 y = x² 에 대입
Y − 3 = (X − 1)2
마이너스는 방향이 반대이기 때문이 아닙니다. 새로운 점의 좌표로 원래 위치를 역추적하는 과정, 즉 이항(移項)이라는 대수적 조작의 자연스러운 결과입니다.

🏷️ 이름표 교체와 최종 확인

이제 마지막 단계만 남았습니다. 임시로 사용했던 대문자 $X, Y$는 이미 역할을 다했습니다. 집합에서 변수명은 의미를 바꾸지 않으며—이를 더미 변수(dummy variable)라 합니다—친숙한 소문자 $x, y$로 이름표를 교체하면 최종 규칙이 완성됩니다.

💡
더미 변수(Dummy Variable)란?

$\{ s \mid s\text{는 정수} \}$와 $\{ t \mid t\text{는 정수} \}$는 정수의 집합으로 완전히 동일한 집합입니다. 집합의 성질은 변수 이름이 아니라 조건에 의해 결정되기 때문입니다. 마찬가지로 $X, Y$를 $x, y$로 교체해도 집합(=그래프)은 변하지 않습니다.

{ (x, y) | y − 3 = (x − 1)2 }

위 슬라이더 값이 그대로 적용된 결과를 아래 그래프에서 확인하세요. 점선이 원래 함수 $y = x^2$이며, 실선이 평행이동된 함수입니다.

"수학은 외우는 것이 아니라 이해하는 것이다."

함수를 점들의 집합으로 바라보는 작은 관점의 전환 하나만으로, 오랫동안 억지로 외워왔던 공식이 지극히 자연스럽고 빈틈 없는 논리의 산물임을 알 수 있습니다.
